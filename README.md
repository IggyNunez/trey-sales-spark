# Trey Sales Spark

A comprehensive sales commission tracking and CRM integration platform.

## Project Info

**Live URL**: https://sales-spark-replica.vercel.app
**Supabase Project**: yrlbwzxphjtnivjbpzsq
**GitHub Repo**: https://github.com/IggyNunez/trey-sales-spark

## Quick Start

```sh
# Clone the repository
git clone https://github.com/IggyNunez/trey-sales-spark.git

# Navigate to the project directory
cd trey-sales-spark

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Technologies

- **Frontend**: React, TypeScript, Vite
- **UI**: shadcn-ui, Tailwind CSS
- **Backend**: Supabase (PostgreSQL + Edge Functions)
- **Deployment**: Vercel

## Features

- Multi-CRM integration (Close, HubSpot, GoHighLevel, Salesforce, Pipedrive)
- Booking platform sync (Calendly, Cal.com)
- Payment processing (Whop, Stripe)
- Real-time commission tracking
- Comprehensive webhook handling
- Dynamic forms and custom metrics
- Role-based access control
